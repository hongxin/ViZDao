# vizdao （应用主体）

ViZDao · 微知道 的前端应用（基于 JetBot 基线改造）。

完整项目说明见上级目录的 [../README.md](../README.md)。

## 开发

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 输出 dist/，纯静态可部署
npm run lint
```

> 当前主视图仍是 JetBot 遗留的聊天/Cosmos 界面，可视化教学核心位于
> [`src/viz/`](src/viz/)（占位骨架），将在头脑风暴与深入规划后落地。
